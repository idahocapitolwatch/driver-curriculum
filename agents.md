# Curriculum Reference Documentation

## Source Documents

The curriculum is based on two Idaho state documents:

- **Idaho Public Driver Education Program Manual** — Contains the content standards and benchmarks (Standards 1.1–9.7), operating policies, student/instructor requirements, and Idaho statutes.
- **DRV101 (Driver Education) Course Map** — Contains the unit and module breakdown, lesson objectives with standard correlations, and assessment mapping.

## Where to Find Curriculum Documentation

- **Scope and sequence overview:** `scope-and-sequence.md` (full standards-to-unit alignment table)
- **Unit overviews:** `course/<unit>/README.md` (standards list, unit objectives, module index)
- **Module details:** `course/<unit>/module-*.md` (standards alignment, lesson objectives, key content topics, assessments)
- **GitHub issues:** Each module has a tracked issue labeled `curriculum` at https://github.com/idahocapitolwatch/driver-curriculum/issues
