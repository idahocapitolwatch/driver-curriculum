## Key Content Topics

### Lane Change Procedures
- Check traffic flow and mirrors
- Signal intention
- Check blind spot (head check)
- Select safe gap
- Smoothly move into new lane
- Cancel signal
- Adjust speed to traffic flow

### Lane Change Legal Requirements
- When lane changes are permitted
- When lane changes are prohibited (solid lines, intersections)
- Signaling requirements
- Right-of-way during lane changes

### Passing Procedures
- Determine if passing is safe and legal
- Check for no-passing zones (signs, markings)
- Check oncoming traffic gap
- Signal and accelerate
- Pass with adequate clearance
- Signal return and check mirror
- Return to lane when safe

### Passing Legal Requirements
- No-passing zones and markings
- Where passing is prohibited
- Speed limits while passing
- Being passed - responsibilities

### Safe Gap Selection
- Judging gaps for lane changes on multi-lane roads
- Judging gaps for passing on two-lane roads
- Factors that affect gap needs (speed, vehicle size, road conditions)
- When NOT to pass

### Blind Area Management
- Mirror check sequence
- Head check (shoulder check) technique
- Timing of checks relative to maneuver
- Blind spot awareness for different vehicle types
